### FroK' Game Engine With DirectX
다이렉트X 공부 및 게임의 통신을 공부하기 위한 레포지토리입니다.
</br>예전에 만들었던 프로젝트를 다시 한 번 코딩해보면서, 게임 엔진의 개념을 복습하며, 동시에 하나의 예시 게임을 만들어 보고 있습니다.
